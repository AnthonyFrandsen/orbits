Week 15 Assignment

Based on this video by Jean Tampon

I know the assignment was supposed to be interacting with the mouse, but I got carried away with using canvas to draw objects.

Click to add an orbiting ball. When balls collide, they turn pink, then fade to green when they don't collide. It seems that eventually the balls will find an orbit where they don't collide!

Use the buttons to clear objects and adjust the speed. The "Solve" button will run the simulation without updating the display until a stable orbit has been reached for a certain amount of time. This does not mean the orbits will forever be stable, but the collisions will be very infrequent.
